The different types of input types in HTML





They are generally used with forms

At least that's how I've been taught to use them so far.

Input Types:





Name of input type What/How it's used Example of it
"Text" text is the default input type
"Password" this type is used when you want to obscure the text as it's entered.
"Number" You use this when you want the user to input a number. Use the step attribute to add up and down arrows.
"Range" USe this when you want the user to input a number within a range. Use min, max, and step attribute to help you.
"Checkbox" Use this when you give multiple options for one answer. Like toppings.
Radio Button Use this when you want to give multiple choices, but only one answer can be chosen.
Dropdown Lists Use this to choose from a plethora of options
"Datalist" Users can enter a value or choose from a list of options
"Textarea Use this for a blog post or anything else that would take up space.
This is a basic overview of the different input types you can use when creating a form
